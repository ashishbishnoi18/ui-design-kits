API Reference
Class Type Default Description
.btn* base — Required base class for all buttons
.btn-primary variant — Solid green background, for primary actions
.btn-secondary variant — Raised background with border, for secondary actions
.btn-outline variant — Transparent with border, for tertiary actions
.btn-ghost variant — No border or background, minimal presence
.btn-danger variant — Red background for destructive actions
.btn-link variant — Looks like a link, no padding or background
.btn-sm size — 30px height, compact padding
.btn-md size default 36px height, standard padding
.btn-lg size — 42px height, generous padding
.btn-icon modifier — Square button for icon-only usage
disabled attribute false Reduces opacity and disables pointer events
