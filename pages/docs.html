Installation

Last updated March 2026

Getting started with NB UI Kit is straightforward. The kit requires no build tools, no package managers, and no frameworks. You simply download the files, include them in your HTML, and start building with the component classes.

Step 1: Download the files

Clone or download the NB UI Kit repository. The project is organized with CSS and JavaScript separated into modular files, with bundled versions for convenience. 

nb-ui-kit/
├── css/
│   ├── nb-all.css          ← bundled CSS (include this)
│   ├── tokens.css
│   ├── reset.css
│   ├── base.css
│   ├── utilities.css
│   ├── components/
│   │   ├── buttons.css
│   │   ├── cards.css
│   │   ├── inputs.css
│   │   └── ...
│   └── layouts/
│       ├── grid.css
│       └── containers.css
├── js/
│   ├── nb-all.js            ← bundled JS (include this)
│   ├── nb-core.js
│   └── components/
│       ├── modal.js
│       ├── dropdown.js
│       └── ...
└── pages/
    └── ...

Step 2: Include CSS

Add the stylesheet to the <head> of your HTML document. The single nb-all.css file includes all tokens, base styles, components, layouts, and utilities. 

<link rel="stylesheet" href="css/nb-all.css">

For better typography, we also recommend loading the Inter and JetBrains Mono fonts from Google Fonts: 

<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700;800&family=JetBrains+Mono:wght@400;500;700&display=swap" rel="stylesheet">

Step 3: Include JavaScript

Add the script tag before the closing </body> tag. The JavaScript handles interactive components like modals, dropdowns, tabs, tooltips, and more. 

<script src="js/nb-all.js"></script>

The script auto-initializes all components on DOMContentLoaded. No configuration needed.

Step 4: Start building

You are now ready to use any NB component. Here is a quick example to verify everything works: 

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My NB Project</title>
  <link rel="stylesheet" href="css/nb-all.css">
</head>
<body>
  <div class="nb-container-xl nb-py-12 nb-text-center">
    <h1 class="nb-text-4xl nb-font-extrabold nb-mb-4">
      Hello, NB UI Kit!
    </h1>
    <button class="nb-btn nb-btn--primary nb-btn--lg">
      Get Started
    </button>
  </div>
  <script src="js/nb-all.js"></script>
</body>
</html>

If you see a bold heading and a green button with a hard offset shadow, you are all set.

CDN

CDN not available yet

A CDN-hosted version of NB UI Kit is planned for a future release. For now, please download the files and host them alongside your project. We will announce CDN availability on our blog and changelog.

Browser Support

NB UI Kit is tested and supported in all modern evergreen browsers. We use standard CSS features including custom properties, @layer, and modern selectors like :where() and :focus-visible.
