Everything you need to build fast

A complete design system with the bold, unapologetic aesthetic your product deserves.

Hard Shadows

Signature offset shadows on every interactive element. No blur, no subtlety -- just bold, clean depth.

Zero Dependencies

Pure CSS and vanilla JavaScript. No frameworks, no build tools, no package managers required.

Accessible

Built with ARIA attributes, focus-visible outlines, and semantic HTML from the ground up.

Responsive

Mobile-first responsive grid and utility classes with sm, md, lg, and xl breakpoints.

Dark Mode Ready

CSS custom properties make it trivial to implement dark mode with a single token override.

Copy & Paste

Every component is a self-contained snippet. Just copy the HTML into your project and go.

Up and running in three steps

No npm install. No config files. Just grab the source and start building.

1

Copy the files

Download the CSS and JS bundles, or clone the repo directly into your project.

2

Add to your project

Link nb-all.css and nb-all.js in your HTML head.

3

Customize tokens

Override CSS custom properties to match your brand colors, fonts, spacing, and shadows.
