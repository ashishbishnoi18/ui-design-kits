# Changelog

All notable changes to the NB UI Kit will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-04-09

### Added
- 141 CSS components with neo-brutalist design
- 71 JavaScript components for interactivity
- 14 page templates (landing, pricing, dashboard, etc.)
- Design token system with `--nb-` prefix
- Dark theme support via `data-theme="dark"`
- CDN distribution via jsDelivr
- npm package distribution
- Component manifest JSON for tooling
- AGENT.md for AI agent consumption

### Components
- Layout: Grid, Container, Stack, Sidebar Layout, Auth Layout
- Forms: Input, Select, Checkbox, Radio, Toggle, File Upload, Tag Input, Range, Search, Date/Time Pickers, Color Picker, Rich Text Editor
- Data Display: Table, Card, Stat Card, Badge, Avatar, Timeline, Calendar, Code Block, JSON Viewer, Tree View
- Navigation: Navbar, Sidebar, Breadcrumb, Pagination, Tabs, Stepper, Dropdown, Command Palette
- Feedback: Alert, Modal, Drawer, Toast, Tooltip, Progress, Spinner, Skeleton, Empty State
- Marketing: Hero, Pricing, CTA, Testimonial, Feature Grid, Newsletter, Trust Bar, Logo Cloud
- Typography: Heading, Text, Code, Blockquote, Mark, Prose
- Decoration: Decoration, Shape, Floating Badge, Highlight Text, Stat Bar, Code Window, Marquee
