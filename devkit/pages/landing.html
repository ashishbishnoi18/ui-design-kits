Everything you need to build

From forms and data tables to navigation and feedback components, DevKit covers the full spectrum.

Layout System

Responsive grid, containers, stacks, and sidebar layouts. Build any page structure with minimal markup.

Form Components

Inputs, selects, checkboxes, toggles, password fields, and search with built-in validation states.

Data Display

Tables, data tables, stat cards, timelines, accordions, and badges for presenting information clearly.

Code Blocks

Syntax-highlighted code display with line numbers, copy buttons, and language badges. Built for dev docs.

Navigation

Navbar, sidebar, tabs, breadcrumbs, pagination, steppers, and dropdowns. Full navigation toolkit.

Feedback & Overlays

Alerts, toasts, modals, drawers, tooltips, spinners, and progress bars for user interaction feedback.

Drop-in simplicity

No build tools. No JavaScript frameworks. Just classes.

html 
<!-- A stat card in three lines -->
<div class="dk-stat-card">
  <div class="dk-stat-card_label">Revenue</div>
  <div class="dk-stat-card_value">$48,290</div>
  <span class="dk-stat-card_trend dk-stat-card_trend--up">+12.5%</span>
</div>

<!-- A responsive grid -->
<div class="dk-grid dk-grid-cols-1 md:dk-grid-cols-3 dk-gap-4">
  <div class="dk-card">...</div>
  <div class="dk-card">...</div>
  <div class="dk-card">...</div>
</div>
