Components / Feedback / Modal

Modal

Centered overlay dialog with backdrop, header/body/footer sections, and size variants.

CSS+JS

Default Modal

Confirm Action

Are you sure you want to proceed? This action cannot be undone.

<div class="dk-modal-backdrop is-open"></div>
<div class="dk-modal is-open" data-dk-modal>
  <div class="dk-modal_header">
    <h3 class="dk-modal_title">Confirm Action</h3>
    <button class="dk-modal_close">&times;</button>
  </div>
  <div class="dk-modal_body">
    <p>Are you sure you want to proceed?</p>
  </div>
  <div class="dk-modal_footer">
    <button class="dk-btn dk-btn--ghost">Cancel</button>
    <button class="dk-btn dk-btn--primary">Confirm</button>
  </div>
</div>
